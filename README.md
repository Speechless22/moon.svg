## svg.moon

<p align="center">
   <img src="https://moon-phase.vercel.app/moon.svg" />
</p>

현재 달의 위상을 보여주는 svg입니다.  
별도 옵션을 주지 않을시 실시간으로 현재 위상을 표현해줍니다.

### 사용법

```html
<img src="https://moon-phase.vercel.app/moon.svg" />
```


### 옵션

```
date: 계산하고자하는 달의 위상, default - 실시간
width: svg 사이즈, default - 100
```
